# Voidly Admin

Minimal admin panel for Voidly.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- TanStack Query

## Requirements

- Node.js 20+
- npm

## Setup

```bash
npm install
```

Create `.env` from `.env.example` and set the backend URL.

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Checks

```bash
npm run lint
npm run prettier:check
```
